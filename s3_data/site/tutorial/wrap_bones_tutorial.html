DesktopHero Tutorial: Wrapping custom assets around existing bones

Step 1: Setup

You will need Blender to export assets for DesktopHero, so install it if you don't already have it.

Add exporter plugin to Blender

We have modified the Three.js exporter for DesktopHero. Download the plugin here.

Unzip the io_three folder, and put it in your Blender installation directory under scripts/addons.


Once the plugin is in the right place, you will need to enable the plugin. Restart Blender and go to File/User Preferences.

Go to the Add-ons tab. In the search bar, type "three" and the plugin named "Import-Export: Three.js Format" should appear. Click the checkbox to the left of the plugin name to enable it. Click "Save User Settings" at the bottom left and exit the window.

If everything worked, you should now have a new exporter under File/Export called "Three.js (.json)".

Download base Blender file

There is a Blender file containing the bone structures that DesktopHero uses. Download it here.

Step 2: Create a DesktopHero asset

Open the base Blender file in Blender.

On the right you can see the different bone structures (armatures) that DesktopHero uses. Underneath each armature are the meshes attached to those bones. You can export these meshes and use them as a base for other designs if you wish.

In this tutorial I will be creating a modified right arm mesh. I hid the other armatures/meshes and made the right arm armature and mesh visible.

Import the new arm. (I created mine in a separate program.)

Move the mesh so that it fits correctly over the bone armature. I hid the default right arm mesh to be able to see mine better.

Apply the object's location. The origin point in Blender (0, 0, 0) will be where your object attaches when it attaches to other bone groups.

On the right menu, click and drag your mesh asset over the armature to set the armature as its parent.

You can weight it how you like, I choose automatic weights here.

If you know how to weight paint to choose how much each vertex is affected by each bone, you can do that. Usually the automatic weights are fine for me.

Make sure the mesh follows the bones as you want it to by switching into bone pose mode and moving the bones around.

If everything works as expected, your mesh is ready to be exported. Go to File/Export and choose Three.js.

The options on the left are the correct options to use in the export. These come with the base file. Save your new mesh.

Step 3: Import the asset into DesktopHero

Open up DesktopHero. Choose the asset you want to replace and click the "Load Mesh" button. Find your mesh and upload it.

If it loaded correctly, you should see a 'success' message.

Your mesh will appear under the 'custom' tab. You can now use it the same as any other DesktopHero mesh.
